На самом деле, сейчас моя структура достаточно глупая. Вот, как она должна выглядеть на самом деле:
QueueOrders
ActiveOrders
Сущность Orders во главе всего. Внутри неё хранится повар, который её выполняет (если выполняет),
курьер, который его доставляет (если доставляет) и общее число, обозначающее прогресс:
- повар не приступил к работе
- повар работает
- не попал на склад
- лежит на складе
- отвозится курьером
- *сущность заказа должна быть уже уничтожена, а её данные - выведены на консоль.*
Tick'ать всех поваров и курьеров всё равно неплохая идея.
